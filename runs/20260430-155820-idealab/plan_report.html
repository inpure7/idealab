요약 특허 상세 시장/전략 상세 기획서 QA MVP 목업
기획서(무엇을 했는지)
  • 제품명: IdeaLab
  • 타겟: 바이브코딩으로 직접 구현하는 1인 창업가, 마이크로 SaaS 포트폴리오를 동시 운영하는 인디 해커, 사이드 프로젝터
  • 문제: 아이디어 생성 속도가 구현 속도를 앞지르고, 수십 개 프로젝트를 Notion·Figma·GitHub 등 파편화된 툴로 관리하다 우선순위를 잃고 아이디어를 사장시킴
  • 솔루션: 아이디어 씨앗 등록 → AI 시장분석 → 목업 자동생성 → MVP 연동 → 버전 관리 → 트래픽 기반 우선순위 추천까지 단일 파이프라인으로 제공하는 귀여운 연구실 컨셉 플랫폼
  • 비즈니스 모델: Freemium SaaS + AI 크레딧 소비 모델. Free 티어로 유입 후 Pro($12/월) 전환 유도. AI 헤비유저는 크레딧 추가 구매로 업셀. 바이브코딩 플랫폼 레퍼럴 커미션으로 부수익.
MVP 기능
  • 아이디어 카드 등록 및 포트폴리오 보드 뷰 (실험체 컨셉, 단계별 상태 표시)
  • AI 기반 1분 시장분석 자동 생성 (타겟·경쟁사·차별점 요약)
  • AI 목업 자동생성 (텍스트 설명 → 와이어프레임 HTML/이미지 출력)
  • 아이디어별 버전 관리 (v1→vN 히스토리, 변경 메모)
  • 서비스별 트래픽 집계 대시보드 (경량 스크립트 한 줄 삽입)
  • 트래픽·버전·업데이트 빈도 기반 우선순위 자동 추천 스코어
  • 바이브코딩 플랫폼 원클릭 연동 (Replit·Lovable URL 연결)
  • 아이디어 공개/비공개 전환 및 외부 피드백 수집 링크
기술 스택
frontend: Next.js + Tailwind CSS (귀여운 연구실 디자인 시스템)backend: FastAPI + Celery (AI 작업 큐)db: PostgreSQL + Redisai: OpenAI API (시장분석·목업 생성)analytics: 자체 경량 SDK + ClickHouse (트래픽 집계)
수익 구조
  • Pro 구독 $12/월 (아이디어 무제한, 목업 20회, 트래픽 무제한, AI 시장분석 10회)
  • AI 크레딧 추가 구매 $5/50크레딧 (목업·시장분석 초과 사용)
  • Team 플랜 $29/월 (공동창업자·스몰팀 2~5인)
  • 바이브코딩 플랫폼 레퍼럴 커미션 (Replit·Lovable 유료 전환 수수료)
  • 화이트라벨 API (액셀러레이터·부트캠프 대상 B2B)
차별점
  • 아이디어 씨앗→목업→MVP→운영까지 단일 파이프라인 (타 툴은 단계별 파편화)
  • 서비스별 실시간 트래픽을 개발 우선순위 신호로 자동 피드백하는 루프 (경쟁사 전무)
  • 귀여운 연구실 컨셉 UX — 아이디어가 '실험체'로 성장하는 스토리텔링으로 감성 차별화
  • 수십~수백 개 마이크로 프로젝트를 각기 다른 버전(v1~vN)으로 병렬 관리하는 포트폴리오 뷰
  • 바이브코딩 플랫폼 원클릭 연동으로 아이디어→구현 브릿지 선점
  • 1인 창업가 전용 설계 — 엔터프라이즈 PM 툴 대비 온보딩 5분 이내
리스크
  • Notion·Linear 등 기존 툴이 AI 기능 추가로 유사 파이프라인 흡수 가능성 (6~12개월 내)
  • 바이브코딩 플랫폼(Replit·Lovable)이 자체 아이디어 관리 기능 내재화 시 브릿지 포지션 약화
  • 1인 창업가 특성상 지불 의사 낮고 무료 툴 선호 — Pro 전환율 3% 이하 위험
  • 트래픽 집계 SDK 설치 허들로 초기 핵심 기능 활성화율 저조 가능성
  • 아이디어 데이터 유출 우려로 민감 아이디어 등록 기피 — 보안 신뢰도 구축 필요
  • AI 목업·시장분석 품질이 기대치 미달 시 핵심 가치제안 훼손
마일스톤

  • Week 4 — MVP 코어 완성: 아이디어 포트폴리오 보드 + AI 시장분석 + 기본 버전 히스토리 기능 동작
  • Week 6 — AI 목업 자동생성 + 경량 트래픽 SDK 베타 출시, 내부 테스터 20명 대상 클로즈드 베타
  • Week 8 — 베타 피드백 반영 및 바이브코딩 플랫폼 연동(Replit URL 연결) 완료, 대기자 300명 확보
  • Week 10 — ProductHunt 공개 론칭, 트위터 빌드 인 퍼블릭 캠페인으로 가입자 500명·Pro 전환 15명 달성
