아이디어
아이디어 인큐베이팅 서비스야. 요즘 바이브코딩을 많이 하니까 1인 창업가가 늘거고, 아이디어 구현할 수 있잖아. 그런데 아이디어가 생기는 속도를 구현이 못 따라가. 그래서 각 아이디어에 대한 기본 정보들을 모아놓고, 목업까지 만들어두는거지. 맘에 들면 실제 구현까지 이어지고. 그런데 MVP 가 끝이 아니더라고 쓰다보면 오류도, 개선해야 할 것도 있지. 수십, 수백개의 아이디어를 분산해서 관리할거야. 예를 들어 A서비스는 v1이고 C서비스는 v5이고. 전체적인 컨셉은 귀여운? 연구실 느낌으로. 나중에는 서비스별 트래픽도 모아서 어느 서비스가 잘 나가니까 우선 개선해야겠다 이런 것도 할거야

  • 아이디어 실험 카드 생성 (제목·문제·솔루션·상태 필드 + 귀여운 연구소 UI)
  • 아이디어별 라이프사이클 상태 관리 (💡아이디어→🎨목업→🚀MVP→⚙️운영)
  • 서비스별 버전 트래킹 (v1~vN 스냅샷 보존 및 변경 이력)

  • LLM 기반 UI 스켈레톤 목업 자동 생성 (Figma JSON 또는 HTML 와이어프레임 출력)
  • 서비스별 트래픽 통합 대시보드 (GA·Plausible·Cloudflare 연동, 성과 점수화)
  • 우선순위 스코어링 엔진 (트래픽·버전·마지막 업데이트 가중치 기반 추천)
  • 전체 아이디어 포트폴리오 뷰 (연구실 보드 — 상태별 카드 그리드)
